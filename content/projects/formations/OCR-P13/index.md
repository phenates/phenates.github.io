Projet n°13 de la formation OpenClassRoom - Administrateur systèmes, réseaux et sécurité.

**Objectif:**

Migrer une application clé vers le cloud. Dans ce projet, vous orchestrerez cette transition en menant d'abord une veille technologique, puis en planifiant la migration pour assurer une continuité de service optimale.

**Compétences acquises dans ce projet:**
- Pilotez un projet d'amélioration du SI
- Migration Cloud
- Assurer une veille technologique sur les différents aspects de l'infrastructure
